A História da Web

voltar

O começo

Tudo começou com as primeiras formas de comunicação não verbal/pessoa, como os telegrafos e radios. Com isso começaram a evoluir devido as necessidades de comunicação, criptografia, calculos e estratégias de guerra durante a primeira e segunda guerra mundial.

História da Internet

A internet começou como um projeto militar nos Estados Unidos, liderado pela ARPA, resultando na ARPANET, a primeira rede de computadores, em 1969.

Os pioneiros

    Década de 40:

    Surgindo os primeiros computadores eletônicos, grades demais para serem transportados ou movidos sozinho, utilizados para fins científicos e militares, por exemplo o ENIAC que era enorme e usavam válvulas.

    Década de 50 e Transitores:

    As Válvulas, nesse periodo são substituídas pelos transitores transformando-os em computadores menores, porém ainda não transportavéis.

    Década de 60:

    Chegada dos circuitos integrados e os transistores reduziram o tamanho e aumentaram a eficiência. Foi uma epoca revolucionaria permitindo assim a criação de computadores mais poderosos e compactos, agora já não precisando ocupar toda uma sala.

A Era dos Microcomputadores

    Década de 1970:

    A Apple I e Apple II marcam a entrada da Apple no mercado de microcomputadores, democratizando o acesso à tecnologia.

    Apple I (1976)

    Kit para montar um computador em casa, com um design minimalista e inovador.

    Apple II (1977)

    Um dos primeiros computadores pessoais de grande sucesso, com gráficos coloridos e um teclado completo.

    Década de 1980:

    A guerra dos computadores pessoais se intensifica, com a Apple lançando o Macintosh em 1984, revolucionando a interface com o usuário com a introdução da interface gráfica e do mouse.

    A Era da Expansão:

    Década de 1990: A Apple enfrenta dificuldades financeiras, mas se recupera com o lançamento do iMac em 1998, um computador colorido e acessível que revitaliza a marca.

Entendendo Clients

O modelo cliente-servidor é fundamental para o funcionamento da internet. O cliente é o dispositivo ou programa que solicita informações, como um navegador da web em um computador ou smartphone.

O que são Servers

voltar

Entendendo Servers

O servidor, por sua vez, é o computador que armazena os dados e os envia ao cliente mediante solicitação.

Por exemplo, quando você acessa um site, seu navegador (cliente) solicita o conteúdo ao servidor onde o site está hospedado. O servidor processa a solicitação e retorna os arquivos necessários para exibir a página.

Introdução ao Html

O que é

Conhecida como Linguagem de Programação de Marcação de Hipertexto é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. "Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

Entendendo Tags

Compostas por sinal de mais e menos (>) (<). iniciado por exemplo "< strong>" e terminado "" tudo que tiver entre elas, serão interpretadas pelo navegador como comandos ou instruções de acordo com w3schools. O ao contrário é com o < sup > que cria texto com sobrescrito ex.: 10x(sem tag) 10x(com tag).

Atributos

Em Html as tags possuem algo chamado Atributos, são normalmente propriedades e informações adicionais compostas por nomes separados que aparecem na tag de abertura, definindo como deverá funcionar e se comportar os elementos dentro da tag. Exemplo: < input type = "button">, < strong id =" titulo">< / strong>

Entendendo Css

CSS é uma linguagem de estilo usada para estilizar páginas web. É uma das principais linguagens da open web e é padronizada em navegadores web de acordo com as especificação da W3C.

O Surgimento

O CSS surgiu em 1996 para resolver o problema da separação entre conteúdo e design. Antes disso, os desenvolvedores dependiam de formatação embutida no HTML, o que dificultava o trabalho em grandes projetos.

Evolução

CSS1: Introduzido em 1996 com propriedades básicas.
CSS2: Adicionou suporte a layouts mais complexos.
CSS3: Trouxe avanços como animações e design responsivo.
OBS.: Mesmo hoje em dia o CSS continua a evoluir, permitindo designs mais sofisticados e melhores experiências de usuário.

Entendendo Sobre linguagens de Programação

"Uma linguagem de programação é um conjunto de regras e sintaxes usadas para criar instruções que um computador pode entender e executar. Elas são fundamentais para desenvolver software, websites, aplicativos móveis, jogos e muitas outras tecnologias. Em outras palavras, uma linguagem de programação é a forma como os desenvolvedores comunicam-se com o computador para que ele realize tarefas específicas." Alguns exemplos de linguagens de programação, como Python, Java, Go, Ruby, C#, JavaScript, C++ e muitas outras, tem seu propósito e características únicas.

Esses três elementos juntos garantem que uma página web seja funcional, bonita e interativa. Enquanto o HTML define a base, o CSS cuida da apresentação, e o JavaScript traz vida à página, permitindo experiências ricas e personalizadas para os usuários.
